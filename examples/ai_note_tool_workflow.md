# Example: AI Note Tool Workflow

## User Input

```text
Forge: 我想做一个帮学生整理课堂笔记的 AI 小工具，先帮我做第一版。
```

## Forge Should Do

1. Clarify only blocking questions: who uses it, input format, output format, platform.
2. Scout lightly: check whether similar note tools or open-source summarizers exist when search is available.
3. Scope MVP:
   - Must do: upload/paste notes, summarize, extract action items, export Markdown.
   - Explicitly not do: login, payments, multi-user sync, mobile app, complex knowledge base.
4. Build plan: simple local web page or CLI first.
5. Definition of Done: user can input one sample note and get a structured Markdown output.

## Expected Output

- MVP plan
- first version file structure
- run command
- manual verification steps
- README outline
