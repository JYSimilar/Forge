# Forge 中文快速调用

## 最常用

```text
Forge: 帮我把这个想法收束成 MVP，并告诉我下一步怎么推进。
```

```text
Forge: 这个项目下一步该干嘛？给我最重要的 3 件事。
```

```text
Forge: 帮我把这件事拆成目标、输入、输出、限制、验收和验证方式。
```

```text
Forge: AI 做完了，帮我验收结果，并告诉我下一步。
```

```text
Forge: 接手这个工程前，先帮我自查当前工作区。
```

## 安全任务单

```text
Forge: 把这个任务写成安全任务单，小步执行，省 token，停在复查关口。
```

```text
Forge: 先给我上下文预算、停手条件和验收标准，不要无限推进。
```

```text
Forge: 在当前 agent 里生成一个角色任务单，要包含验收标准、上下文预算和不要碰的范围。
```

## 小任务

```text
只给我这次提交的 commit message，不用解释。
```

```text
这个是小任务，直接给我结果和一个验证步骤。
```

## 已有工程

```text
Forge: 接手这个工程前，先帮我自查当前工作区。
```

```text
Forge: 这个文件夹里有没有项目？如果有，告诉我技术栈、风险和下一步。
```

```text
Forge: 检查这个项目放生产会不会炸，重点看错误处理、日志、输入校验、并发安全和测试。
```

## 验收和提交

```text
Forge: AI 做完了，帮我按完成标准验收一下，不合格就给返工提示词。
```

```text
Forge: 帮我看这次改动能不能提交。
```

```text
Forge: 帮我生成 commit message 和 PR 描述。
```

## 文档和交付

```text
Forge: 帮我写一份别人也能看懂的 README。
```

```text
Forge: 帮我写 Quick Start，要求普通人能照着跑起来。
```

```text
Forge: 我要把项目交给别人，帮我写一份交接文档。
```

## 高级功能

```text
Forge: 给我同时生成人类索引和机器索引。
```

```text
Forge: 跑一轮 Forge Doctor 稳定性总检，检查这版能不能发布。
```

```text
Forge: 在当前 agent 里按前端、后端、测试几个角色分工。
```
