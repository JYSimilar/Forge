# Forge English Quick Commands

## Start Here

```text
Forge: Turn this idea into a safe work order for the current agent.
```

```text
Forge: What are the 3 most important next steps for this project?
```

```text
Forge: Split this into goal, input, output, limits, acceptance, and verification.
```

```text
Forge: The AI finished. Verify the result and tell me the next step.
```

```text
Forge: Audit this workspace before we change anything.
```

## Safe Work Order

```text
Forge: Write this as a Safe Work Order: small step, low token, stop at Review Gate.
```

```text
Forge: Give me the context budget, stop condition, and acceptance criteria first.
```

```text
Forge: Create a current-agent role work order with acceptance, context budget, and forbidden scope.
```

## Small Tasks

```text
Only give me the commit message. No explanation.
```

```text
This is a small task. Give me the result and one verification step.
```

## Existing Projects

```text
Forge: Audit this workspace before we change anything.
```

```text
Forge: Does this folder contain a project? If yes, summarize stack, risks, and next steps.
```

```text
Forge: Check whether this project would break in production. Focus on errors, logs, input validation, concurrency, and tests.
```

## Review and Submit

```text
Forge: The AI finished. Verify against acceptance criteria and write a rework prompt if needed.
```

```text
Forge: Review whether these changes are ready to submit.
```

```text
Forge: Write a commit message and PR description.
```

## Docs and Handoff

```text
Forge: Write a README that real users can understand.
```

```text
Forge: Write a Quick Start that a normal user can follow.
```

```text
Forge: Write a handoff document for someone else to continue the project.
```
