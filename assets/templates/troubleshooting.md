# 故障排查表

| 问题 | 可能原因 | 解决办法 |
|---|---|---|
| 启动失败 | 依赖未安装 / 配置错误 | 检查依赖和配置文件 |
| 端口被占用 | 已有服务占用端口 | 修改端口或关闭占用进程 |
| 配置文件找不到 | 当前运行目录不对 / 路径写错 | 使用绝对路径或环境变量 |
| API key 不生效 | 环境变量没加载 / key 错误 | 重新设置环境变量并重启 |
| 外部服务连不上 | 服务未启动 / 地址错误 / 网络问题 | 检查服务状态和地址 |
| 测试失败 | 依赖、配置或逻辑变化 | 查看失败用例和错误堆栈 |
| Docker 运行失败 | 镜像构建失败 / 端口映射错误 | 查看 docker logs |
